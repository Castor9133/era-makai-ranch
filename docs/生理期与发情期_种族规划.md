# 生理期与发情期系统 — 种族差异化规划（远期）

> **定位**：长线设计文档，**不要求近期实现**。与「榨乳口便器牧场」等玩法**可解耦**；实现时以 **CONFIG 总开关** + **种族规则表** 为轴逐步落地。  
> **现状**：工程内仅有占位（如 `FLAG:5012`、`CFLAG:939` 日进等），本文描述**目标体验与数据/规则分层**，便于日后开专项迭代。

---

## 1. 设计目标

1. **种族差异化**：同一套「日进计数 + 阶段枚举」框架下，用 **规则表** 决定是否有周期、周期长度、阶段效果倍率、是否启用发情期等。  
2. **人类在魔界的特殊性**：**日历层**采用下列 **15 日短周期**（见 §3.1）；叙事与数值上仍可保留「魔界适应」——**症状强度、体力/情绪惩罚等 ×0.4～0.5（可 CONFIG）**，与「现实地球女性日历」不必同形。  
3. **仙人 / 吸血鬼**：**无生理期**（或永久记为「不适用」阶段）；可选 **叙事占位**（如「灵力周流」「血渴节律」）与数值系统分离，避免与妊娠逻辑打架。  
4. **动物类亚人**：**双轨** ——  
   - **生理期**：策划语「常一点」建议定稿为二选一：**① 周期更长**（登场频率低、单次持续信息量少，例如 32～40 日游戏日一圈）；或 **② 阶段更「日常化」**（症状曲线平缓、但日历上仍按 28 日类基准）。下文默认 **① 周期长**；若选②，在规则表里改 `周期长度` 与 `阶段强度` 列即可。  
   - **发情期**：周期**更短**、窗口**更集中**（例如每年数次或按短周期嵌套），**性欲／服从／特定 COM 倍率**等偏高，与生理期可 **错开、叠加、或互斥高峰**（由规则表列明，叠加时建议 **封顶** 避免数值爆炸）。  
5. **魔族／天界种族**：**在人类（当前模板）基础上整周期加倍**——周期总长、经期长、危险期长等 **同比例 ×2**；生理参照仍以**现实中人类女性**各阶段比例为基准，再整体放大到游戏日历（见 §3.2）。  
6. **精灵／半精灵**：以**现实中人类女性**典型周期为基准；**月经持续日数不变**（不加倍）；**危险期、黄体期**等策划划定阶段的**持续日数再加倍**，卵泡期／周期总长由表重算（见 §3.3）。  
7. **其他类人族**：**矮人族、半身人、半吸血鬼、半兽人（人类混血侧）** 等凡属「**以人类血统或人类型躯体为主**」的类人，**生理期日历与 §3.1 人类（魔界）完全一致**（15 日、经期 3、危险期 4 等同段）；**症状强度**默认与人类相同规则（含魔界减半是否适用，可由 `TALENT`／CONFIG 再分）。**纯血吸血鬼**仍按 §1 第 3 条 **无生理期**；**半吸血鬼**走类人规则（有周期，同人类表）。  
8. **其他**：动物亚人发情期等仍按 §1 第 4 条；仙人／纯吸血鬼无周期仍适用。

## 2. 概念区分（实现时再命名 CFLAG）

| 概念 | 用途（规划） |
|------|----------------|
| **生理期** | 子宫内膜周期叙事 + 可选体力/痛觉/情绪 **LOSEBASE** 或 **PALAM 获得倍率** 修正；与 **妊娠** 互斥或暂停计数需单独规则。 |
| **发情期** | **性行为动机与快感侧** 修正（淫欲、特定奉仕 SOURCE、日程「烦躁」等），**不等于**生理期；动物系可两轨并存。 |
| **无周期** | 计数器冻结或恒为「阶段 0」，日结 **不推进**；口上可走 **空分支** 或种族专属台词。 |

建议日后为发情期单独预留计数（当前 `CFLAG:940` 占位可指向「发情周期日／发情阶段」等，**实现时再定名**）。

## 3. 日历层定案（策划数值）

实现时用 **`cycle_length` + 各阶段 `[start,end)` 日偏移`** 驱动计数器即可；下表为 **定案意图**，与妊娠「排卵／易孕」脚本对齐时允许微调边界 **±1 日**。

### 3.0 医学上常见的「时间顺序」（与 §3.1 编号的关系）

教材与临床上常用的 **时间先后** 是：

**月经期 →（早卵泡与经期重叠后）卵泡期 → 排卵期 → 黄体期 →（黄体退化）经前期／下次月经来潮**

- **周期第 1 天**：多取 **月经来潮的首日**（并非排卵日）。  
- **危险期／易孕窗口**：通常 **包围排卵日前后**（属卵泡期末至排卵后极短窗 + 可含黄体初的争议段，游戏内可简化为「排卵前后连续 N 日」）。  
- **黄体期**：排卵之后至下次月经前，**末尾**可合并 **经前期（PMS）** 叙事。

**§3.1 的 15 日表** 为 **游戏用环状排期**（把「月经期」摆在 **第 7～9 日** 正中），**不是**「第 1 天 = 月经首日」的临床编号；因此 **第 1～6 日绝不能标成卵泡期**——在真实顺序里它们处在 **上一轮黄体末期 + 经前期**（月经来潮之前）。**第 10～13 日** 才对应 **排卵前后 ≈ 易孕危险期**（紧接本次月经结束后的 **卵泡晚期 + 排卵** 的**压缩**模型；15 日总长偏短，**不追求**与 28 日教科书逐日等价，仅保持 **先后关系自洽**）。**第 14～15 日** 为 **黄体期**（短）。

若日后改为 **「第 1 天 = 月经第 1 天」的线性表**，应 **整表重划** 阶段边界；本文 §3.2 魔族／天界对人类模板的 **倍长映射** 仍以 §3.1 **当前环位** 为准。

### 3.1 人类（魔界）：15 日一周期

| 项目 | 定案 |
|------|------|
| **周期总长** | **15** 游戏日（模 15）。 |
| **月经期** | 连续 **3** 日，置于本环 **正中**：**第 7～9 日**（自 1 起算，含首尾共 3 日）。 |
| **危险期（易孕／排卵前后）** | 连续 **4** 日；与月经 **不重叠**。**第 10～13 日**（紧接月经后 → **卵泡晚期 + 排卵前后** 的易孕窗，游戏压缩）。 |
| **黄体期** | **第 14～15 日**（2 日；排卵后至本轮环末的短黄体）。 |
| **黄体末／经前期** | **第 1～6 日**（6 日；**非**卵泡主段——为 **上一轮黄体退化 + 经前** 至本次月经来潮）。合计 6+3+4+2=15。 |
| **说明** | 与 §3.0 对照：顺序为 **…黄体末/经前(1～6) → 月经(7～9) → 排卵前后易孕(10～13) → 黄体(14～15) →** 回到下一环第 1 日。若排卵逻辑要钉死 **排卵日**，建议落在 **第 11 或 12 日**（可 ±1 CONFIG）。 |

### 3.2 魔族、天界种族：在人类模板上「整周期加倍」

| 项目 | 定案 |
|------|------|
| **参照基准** | 以 **§3.1 人类（15 日）** 为模板，对照**现实中人类女性**「经期 / 易孕 / 黄体」的**相对比例**，再整体放大。 |
| **周期总长** | **30** 游戏日（= 15×2）。 |
| **月经期** | **6** 日（= 3×2）；与 §3.1 **一一对应**：人类第 **7～9** 日 → 魔族／天界第 **13～18** 日（每人类日展开为连续 2 游戏日）。 |
| **危险期** | **8** 日（= 4×2）；人类第 **10～13** 日 → 魔族／天界第 **19～26** 日。 |
| **黄体末／经前 + 黄体（序）** | 人类第 **1～6** 日 → 第 **1～12** 日（12 日）；人类第 **14～15** 日 → 第 **27～30** 日（4 日）。合计 12+6+8+4=**30**。（命名与 §3.0 一致，**非**「卵泡占 1～6」。） |
| **其余** | **症状强度**可不套用人类魔界减半（本土生物），另列 CONFIG。 |

### 3.3 精灵、半精灵：现实女性基底 + 经期不变 + 危险期／黄体期加倍

| 项目 | 定案 |
|------|------|
| **适用范围** | **精灵**与**半精灵**共用本段规则（若半精灵需弱化，再以 `TALENT` 或子类覆盖表减成）。 |
| **参照基准** | **现实中人类女性**教科书式周期（常用 **28 日** 或医学常用分段）；**不以 §3.1 的 15 日人类短周期**为精灵基底，避免与「现实比例」语义冲突。 |
| **月经持续日数** | 与现实中人类女性常用取值 **相同**（例如 **5 日**，或采用工程内统一常量）；**不对「月经天数」做加倍**。 |
| **危险期、黄体期等** | 在基底表上，将**危险期、黄体期**（及策划列入「非月经期且需加倍」的阶段）的 **持续日数 ×2**；**卵泡期**长度若未加倍则保持基底日数，**周期总长** = 月经 +（加倍后各段）+ 卵泡，**必大于 28**。 |
| **示例（示意）** | 将 28 日拆成 **时间上顺接** 的四段（命名顺序须与 §3.0 一致）：**月经 5** → **卵泡 5** → **危险／排卵前后 6** → **黄体 12**（5+5+6+12=28）。加倍仅作用于 **危险期、黄体期** → 危险 **12**、黄体 **24**；**月经仍为 5**、**卵泡仍为 5** → 总长 **5+5+12+24=46** 游戏日；排卵／易孕判定表按 **46 日模**重划边界。（先前文档把四段名字写乱序，易误解为「卵泡在黄体后」——已更正。） |

### 3.4 类人族（与人类共用 §3.1）

下列种族**不单独造日历**，实现时归入 **`MENSTRUAL_RULE_HUMAN_15`**（名称示意）即可：

- **矮人族、半身人**（与半身人同类的「小人族」分支若存在，策划可并入本行同一规则）  
- **半吸血鬼**（纯血吸血鬼は無周期／§1 第 3 条）  
- **半兽人** 等以 **人类混血／人类型月经解剖** 为默认叙事的种族  
- 其他策划指定为「**参照人类即可**」的类人标签

**例外**：若某角色同时带 **精灵／半精灵** 血统表记，则 **优先 §3.3**（不与 §3.1 混用）。

### 3.5 种族规则矩阵（摘要）

| 种族/标签（示例） | 周期模板 | 备注 |
|-------------------|-----------|------|
| **人类**（魔界） | §3.1，15 日；症状可减半 | 危险期 4 日、月经 3 日。 |
| **类人族**（矮人、半身人、半吸血鬼、人类侧半兽人等） | **同 §3.1** | 与人类同一 `cycle_length` 与阶段边界；口上・体质可再分。 |
| **魔族、天界种族** | §3.2，30 日；经期 6、危险 8 | 整相对人类模板 ×2。 |
| **精灵、半精灵** | §3.3，现实 28 式基底 | 月经天数不变；危险期／黄体期等 **日数加倍**。 |
| **仙人、纯吸血鬼** | 无 | 同 §1 第 3 条。 |
| **动物类亚人** | 另轨：生理期 + 发情期 | 同 §1 第 4 条。 |

**种族判定落点（实现时查阅）**：优先复用现有 **`TALENT`／角色作成 `CSTR`／种族 BIT**；若无单一字段，再用 **`CFLAG` 位图「规则覆盖 ID」**（避免在文档里写死未存在的 ID）。

## 4. 与现有/未来系统的接口（仅清单）

- **妊娠**：经期是否禁止部分 COM、是否暂停排卵逻辑、魔界人类「减半」是否影响受孕率 —— **需单独 CONFIG 或子规则**。  
- **泌乳**：与 **药催／魔催** 线并存时，建议 **发情期** 微调喷乳欲望、**生理期** 仅轻量影响体力与痛阈，避免与 `MARK:母乳体质` 强耦合。  
- **调教 SOURCE**：阶段 → `TIMES` 痛苦/耻辱/淫欲 等 **分渠道** 修正；**封顶**写在统一 `#FUNCTION`。  
- **排泄羞耻**：腹痛与膀胱不适 **叙事分流**（已实现膀胱链的可接「经前腹胀」占位）。  
- **榨乳口便器线**：贞操带／尿道塞在特定阶段是否 **CONFIG 禁止** —— 远期再接。

## 7. 与当前代码占位的关系

- 现有 **`RANCHORAL_MENSTRUAL_DAILY_TICK`** 仅为 **模 28 均匀日进**，**未**体现种族；日后应改为 **`TRYCALL` 统一入口** 内按种族 **`cycle_length`（15 / 30 / 46 等）** 取模推进。
