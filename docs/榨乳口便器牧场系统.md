# 榨乳口便器牧场系统（实现摘要）

> 与总设计计划（Cursor 计划「榨乳口便器牧场设计」）对齐；**药／魔催泌乳与Ｖ处女并存**，不采用处女妊娠产奶。本文件描述**已接线**的工程行为与数据位。

## 总开关

| FLAG | 说明 |
|------|------|
| `FLAG:5011` 牧场口便器玩法启用 | `0`：跳过本系统全部钩子（旧档默认）；`1`：启用。可在 `CONFIG_2500～2599,システム設定.ERB` 中切换。 |

## 奴隶 CFLAG（`Cflag.csv`）

| ID | 名称 | 范围／语义 |
|----|------|----------------|
| 933 | 口便器适性 | 0～1000，奉仕类指令缓慢成长 |
| 934 | 栓具习惯日 | 0～500，调教内有耗时指令且装着肛塞或尿道塞时累加 |
| 935 | 绝顶服从练度 | 0～1000，与「无绝顶调教回合」挂钩 |
| 936 | 排尿许可模式 | 0 默认；预留闸门位 |
| 937 | 口腔习惯练度 | 0～1000 |
| 938 | 绝顶寸止连续达成 | 连续多回合调教结算无绝顶则 +1，否则衰减 |
| 939 | 生理周期日 | 0～27，`FLAG:5012` 开启时日进 |
| 940 | （预留） | 占位，可与体质位图联动 |

## 调教内钩子

- **`RANCHORAL_TRAIN_POST_COMMAND`**（`●SOURCE_CHECK.ERB` 在 `SOURCE_UP` 之后调用）  
  - 计入 **口便器适性／口腔习惯**（口・喉系 `SELECTCOM` 集）。  
  - 计入 **非插入奉仕**（素股 50、乳交 53～56、手交 57～58、深喉 52、进阶口 132～133）。  
  - **栓具习惯**：本回合 `SOURCE:已用时间` 且 `TEQUIP:肛塞` 或 `TEQUIP:尿道塞` 时 `CFLAG:934` 微增。

## 精神 SOURCE 倍率

- **`GET_RANCHORAL_TRAIN_SOURCE_MUL`**（`#FUNCTION`）：在 `○SOURCE_UP.ERB` 对 **`SOURCE:恭顺` `SOURCE:屈服`** 在调教中施加 `TIMES … /1000`（上限约 +12%），随 `口便器适性` 与 `栓具习惯日` 升档。

## 搾乳日程

- **`RANCHORAL_LACTATION_SESSION_BONUS`**（`スケジュール搾乳.ERB` 的 `ENDSELECT` 之后）：当 `口便器适性` 达阈值且本次有喷乳时，微量增加 **恭顺／屈服 PALAM**（日程结算路径）。

## 调教结束：绝顶管理（轻量）

- **`RANCHORAL_TRAIN_SESSION_END_ORGASM_TRACK`**（`●調教COMMON.ERB` 在 `TRAIN_END_RESULT` 之后）：若本会话 `EX:总绝顶 == 0` 且指令数 ≥4，则 `CFLAG:938` 连续计数 +1 并涨 `绝顶服从练度`；否则衰减。需 `FLAG:5013` 绝顶管理启用。

## 日结：生理期占位

- **`RANCHORAL_MENSTRUAL_DAILY_TICK`**（`◯排泄羞耻_膀胱.ERB` 的 `EXCRETION_DAILY_ALL` 每人循环末）：`FLAG:5012` 开启时 `CFLAG:939` 日进模 28。  
- **种族差异化与发情期**（远期规划，未实现）：见 [`生理期与发情期_种族规划.md`](生理期与发情期_种族规划.md)（人类 **15** 日模板；**矮人、半身人、半吸血鬼** 等类人同模板；魔族／天界 **30** 日；精灵／半精灵 **46** 日示例等）。

## 成就：奶牛牧场称号

- **里程碑 FLAG `933`**（`ERB/マイルストーン/マイルストーン_920～.ERB` 内 `MILESTONE_CHECK_933`）：条件示例——`CMATCH(CFLAG:乳牛,1) ≥ 3`、`FLAG:母乳出货总量 ≥ 30000`、`FLAG:魔界等级／奶农业 ≥ 3`。达成时与其它里程碑相同贡献 PT 奖励。
- **牧场情報**：`RANCH_STATUS_MONEY` で `FLAG:933 > 0` のとき称号一行を表示。
