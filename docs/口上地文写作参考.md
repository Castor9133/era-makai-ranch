## 一、T1「规矩·羞耻展演」——《白玉道》

### 1.1 核心权力关系
- 奴/主子结构：称呼链必须明确，敬语与自贬交替
- 亲密永远伴随 **职务/罚则/围观** 语境——不存在"没人在场就甜"的切换
- 身体是 **检视对象、标记载体、器具接口**，不是亲密对象

| 维度 | 示例词汇 |
|------|---------|
| **身体部位** | 膣穴、后窍、口穴、肉唇、淫核、尻肉、肛褶、尿道口、喉奥、乳首、乳肉 |

| **心态** | 不敢动、咬着唇数数、僵着背让人看、想夹又不敢夹、浑身发烫还要答话 |
| **权力语** | 「回主人…」「请主人验过…」「奴婢知错了…」「全凭主人处置…」 |

### 1.3 句式节奏
```
场面铺陈（3-5句，交代围观/职务语境）
  → 规则台词（权力宣告/验身口令）
    → 羞耻升级（身体展开/被看/被碰/被评价）
      → 服从/嘴硬二择（身体诚实 vs 嘴上抵抗）
```

**示例地文**（露骨版）：
```

```

### 1.4 适用场景
- 调教命令执行（高屈辱/高支配场合）
- 公开调教/围观语境
- 洗澡羞辱线（冲洗/验身）

### 1.5 禁止

### 10.5 口上写作禁忌

1. **不要**用现代言情词汇（"宠溺"、"温柔乡"、"霸道"）

4. **不要**机械重复同一句式——同场景至少3-5种不同说法
5. **不要**吝啬 `\n`——长句超过25字必须换行
6. **不要**写满屏「……」——省略号用三次就够了

## 十四、哥布林捕获·种族专属写作指南

### 14.1 核心设定
- **哥布林 ≠ 人类**——它们的性行为更接近**捕食/标记/繁殖**，而非色情或调教
- **种族特征**：矮小（1.2-1.4m）、绿皮/灰皮、尖耳獠牙、体味重（酸腐/泥腥/血腥混合）、语速快嗓音尖
- **社会结构**：群体行动（3-8只一队）、服从强者、没有一夫一妻概念
- **对俘虏的态度**：战利品/繁殖容器/肉便器——用完可能卖掉也可能吃掉
- **关键区别于人类轮奸**：
  - 人类T9轮奸有"堕落过程"——哥布林不关心你的心理，它们只关心"能不能用"
  - 哥布林不会"调教"——行为模式是本能驱使，不是有计划的驯化
  - 哥布林会**咬/抓/用牙齿固定**——这些是人类轮奸中没有的
  - 哥布林的精液量极大且味道刺鼻（绿白色/腥臭）——它们会用精液"标记"俘虏

### 14.2 叙事风格对照
