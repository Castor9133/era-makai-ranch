## 2. 男女互动 · 抽象维度总结

### 2.1 《白玉道》→ T1

- **权力差**：奴/主子/军规/验身话术与身体控制同屏；亲密常伴随**职务、罚则、围观**。
- **称谓**：链式敬语与自贬并存；柔媚表层下可藏**刚性反击**（语锋或行动转折）。
- **节奏**：场面铺陈 → 规则台词 → 羞耻升级；身体描写偏**检视、标记、器具/姿态**。
- **身体描写粒度**：公共性、展演性大于私密抚慰；适合 **SOURCE 屈辱/支配** 加权语境。

### 2.2 《龙魂侠影》→ T2

- **权力差**：江湖身份与情感张力并行；男女互动常呈 **羞（不信/要哭/推拒）→ 占有欲/体温上升 → 直白部位描写**。
- **对白密度**：对话驱动推进的比例高，宜 **短句互驳 + 动作接续**。
- **身体描写粒度**：触感与体积感直白，但仍需用**原创**短距意象，忌复用原著专有表述。
- **节奏**：情绪一句、动作一句交替，避免长段说明书式旁白。

### 2.3 《13216》保姆线 → T3

- **权力差**：感恩、救助、同住缩小心理距离；**自卑**与「值不值得被接纳」推动献身冲动。
- **称谓**：由疏远称呼向直呼名过渡可作为**段落台阶**。
- **催化剂**：仪式感（生日等）、饮酒、脆弱时刻；男视角可带**生活细节观察**。
- **身体描写粒度**：从衣料、呼吸、距离感入手，适合 **拜访短时轻 H**。
