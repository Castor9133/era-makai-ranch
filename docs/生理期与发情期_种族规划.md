# 生理期与发情期系统 — 种族差异化规划（远期）

> **定位**：长线设计文档，**不要求近期实现**。与「榨乳口便器牧场」等玩法**可解耦**；实现时以 **CONFIG 总开关** + **种族规则表** 为轴逐步落地。  
> **现状**：工程内仅有占位（如 `FLAG:5012`、`CFLAG:939` 日进等），本文描述**目标体验与数据/规则分层**，便于日后开专项迭代。

---

## 1. 设计目标

1. **种族差异化**：同一套「日进计数 + 阶段枚举」框架下，用 **规则表** 决定是否有周期、周期长度、阶段效果倍率、是否启用发情期等。  
2. **人类在魔界的特殊性**：**日历层**采用下列 **15 日短周期**（见 §3.1）；叙事与数值上仍可保留「魔界适应」——**症状强度、体力/情绪惩罚等 ×0.4～0.5（可 CONFIG）**，与「现实地球女性日历」不必同形。  
3. **仙人、女神、纯吸血鬼、妖狐**：**无月经／无内膜周期**；叙事上为 **灵力、神力、血统或妖力自控**（可压制或本无人类式月经）；**不推进**月经计数器；可选独立剧情节律（血渴等）与数值 **解耦**。  
4. **龙人、翼人、动物类亚人**（**不含精灵**，见 §1 第 7 条）：共用 **§3.3「年周期 + 发情」轨**（定案）：**月经每次连续 7 日、每年两次**；非月经日在 **普通日／发情期** 上按 **2:1** 推进——**30 日一子环：普通 20 日、发情 10 日**（模 30 循环）。**发情期**内：调教、同房、洗澡等 **更易湿润**，**受孕判定加权**。**普通日**内：**交配／同房不触发怀孕**（妊娠入口关闭或权重 0，CONFIG 可开例外）。  
5. **魔族、天使、天人**：**彼此共用同一套日历**（见 §3.2），即在 **§3.1 人类（15 日）** 模板上 **整周期 ×2**（30 日、经期 6、易孕窗 8 等）。**女神**不适用 §3.2（见 §1 第 3 条无月经）。  
6. **其他类人族**：**矮人族、半身人、半吸血鬼、半兽人（人类混血侧）** 等凡属「**以人类血统或人类型躯体为主**」的类人，**生理期日历与 §3.1 人类（魔界）完全一致**；**半吸血鬼**有周期、同人类表。  
7. **精灵类（全体一致）**：**高等精灵、半精灵、暗精灵** 等凡标为精灵血统者，**共用 §3.4**——先按 **现实中人类女性** 列出一环 **28 日** 各段（其中 **月经 7 日**）；**除月经以外的每一段日历时间全部 ×2**（非月经合计 **21** 日 → **42** 日），**精灵一环总长 = 7+42 = 49 日**（见 §3.4）。**不与 §3.1 类人、不与 §3.3 亚人发情轨混用**。

## 2. 概念区分（实现时再命名 CFLAG）

| 概念 | 用途（规划） |
|------|----------------|
| **生理期** | 子宫内膜周期叙事 + 可选体力/痛觉/情绪 **LOSEBASE** 或 **PALAM 获得倍率** 修正；与 **妊娠** 互斥或暂停计数需单独规则。 |
| **发情期** | **性行为动机与快感侧** 修正（淫欲、特定奉仕 SOURCE、日程「烦躁」等），**不等于**生理期；动物系可两轨并存。 |
| **无周期** | 计数器冻结或恒为「阶段 0」，日结 **不推进**；口上可走 **空分支** 或种族专属台词。 |

建议日后为发情期单独预留计数（当前 `CFLAG:940` 占位可指向「发情周期日／发情阶段」等，**实现时再定名**）。

## 3. 日历层定案（策划数值）

实现时用 **`cycle_length` + 各阶段 `[start,end)` 日偏移`** 驱动计数器即可；下表为 **定案意图**，与妊娠「排卵／易孕」脚本对齐时允许微调边界 **±1 日**。

### 3.0 医学上常见的「时间顺序」（与 §3.1 编号的关系）

教材与临床上常用的 **时间先后** 是：

**月经期 →（早卵泡与经期重叠后）卵泡期 → 排卵期 → 黄体期 →（黄体退化）经前期／下次月经来潮**

- **周期第 1 天**：多取 **月经来潮的首日**（并非排卵日）。  
- **危险期／易孕窗口**：通常 **包围排卵日前后**（属卵泡期末至排卵后极短窗 + 可含黄体初的争议段，游戏内可简化为「排卵前后连续 N 日」）。  
- **黄体期**：排卵之后至下次月经前，**末尾**可合并 **经前期（PMS）** 叙事。

**§3.1 的 15 日表** 为 **游戏用环状排期**（把「月经期」摆在 **第 7～9 日** 正中），**不是**「第 1 天 = 月经首日」的临床编号；因此 **第 1～6 日绝不能标成卵泡期**——在真实顺序里它们处在 **上一轮黄体末期 + 经前期**（月经来潮之前）。**第 10～13 日** 才对应 **排卵前后 ≈ 易孕危险期**（紧接本次月经结束后的 **卵泡晚期 + 排卵** 的**压缩**模型；15 日总长偏短，**不追求**与 28 日教科书逐日等价，仅保持 **先后关系自洽**）。**第 14～15 日** 为 **黄体期**（短）。

若日后改为 **「第 1 天 = 月经第 1 天」的线性表**，应 **整表重划** 阶段边界；本文 §3.2（魔族、天使、天人）对人类模板的 **倍长映射** 仍以 §3.1 **当前环位** 为准。

### 3.1 人类（魔界）：15 日一周期

| 项目 | 定案 |
|------|------|
| **周期总长** | **15** 游戏日（模 15）。 |
| **月经期** | 连续 **3** 日，置于本环 **正中**：**第 7～9 日**（自 1 起算，含首尾共 3 日）。 |
| **危险期（易孕／排卵前后）** | 连续 **4** 日；与月经 **不重叠**。**第 10～13 日**（紧接月经后 → **卵泡晚期 + 排卵前后** 的易孕窗，游戏压缩）。 |
| **黄体期** | **第 14～15 日**（2 日；排卵后至本轮环末的短黄体）。 |
| **黄体末／经前期** | **第 1～6 日**（6 日；**非**卵泡主段——为 **上一轮黄体退化 + 经前** 至本次月经来潮）。合计 6+3+4+2=15。 |
| **说明** | 与 §3.0 对照：顺序为 **…黄体末/经前(1～6) → 月经(7～9) → 排卵前后易孕(10～13) → 黄体(14～15) →** 回到下一环第 1 日。若排卵逻辑要钉死 **排卵日**，建议落在 **第 11 或 12 日**（可 ±1 CONFIG）。 |

### 3.2 魔族、天使、天人：在人类模板上「整周期加倍」（彼此共用）

| 项目 | 定案 |
|------|------|
| **适用范围** | **魔族**、**天使**、**天人** 共用本表同一 `cycle_length` 与阶段边界。**女神**不适用（无月经，§1 第 3 条）。 |
| **参照基准** | 以 **§3.1 人类（15 日）** 为模板，对照**现实中人类女性**「经期 / 易孕 / 黄体」的**相对比例**，再整体放大。 |
| **周期总长** | **30** 游戏日（= 15×2）。 |
| **月经期** | **6** 日（= 3×2）；与 §3.1 **一一对应**：人类第 **7～9** 日 → 本轨第 **13～18** 日（每人类日展开为连续 2 游戏日）。 |
| **危险期** | **8** 日（= 4×2）；人类第 **10～13** 日 → 第 **19～26** 日。 |
| **黄体末／经前 + 黄体（序）** | 人类第 **1～6** 日 → 第 **1～12** 日（12 日）；人类第 **14～15** 日 → 第 **27～30** 日（4 日）。合计 12+6+8+4=**30**。（命名与 §3.0 一致，**非**「卵泡占 1～6」。） |
| **其余** | **症状强度**可不套用人类魔界减半，另列 CONFIG。 |

### 3.3 龙人、翼人、动物类亚人：年两次月经 + 普通日／发情期（2:1 子环）

> **说明**：本轨仅 **龙人、翼人、兽耳等动物类亚人**。**精灵类**一律走 **§3.4**，勿并入本条。

| 项目 | 定案 |
|------|------|
| **月经** | 每次连续 **7** 游戏日；**每年触发两次**（两个 `月经首日` 锚点，建议相隔约 **半年** 或按绝对日数表驱动，**±1～数日 CONFIG** 可接受）。 |
| **普通日／发情期（子循环）** | 在非月经日的日历推进上，使用 **30 游戏日一环**（模 30）：**第 1～20 日 = 普通日**，**第 21～30 日 = 发情期**（**普通:发情 = 20:10 = 2:1**）。每过 1 游戏日，子环索引 **+1**（遇月经日的处理见下行）。 |
| **月经与子环重叠** | 建议 **月经日优先**：当日按 **月经** 判定叙事与妊娠互斥；子环索引 **暂停** 或 **仍递增但不判为发情** 二选一，须在实现常量表注明，避免「月经第 1 天误算成发情末段」。 |
| **发情期** | **调教、同房、洗澡** 等涉及性刺激或亲密接触的场合：**湿润相关数值上升**（润滑、分泌、敏感等），**怀孕判定加权**（易受孕）。 |
| **普通日** | **交配／同房不怀孕**（妊娠判定不通过或权重 0；魔法／道具受孕等另走独立入口）。 |
| **与动物亚人** | **猫耳、犬耳等兽耳族**（不含种族标签为 **妖狐** 者）与 **龙人、翼人** 共用本轨。**妖狐** 见 §1 第 3 条与矩阵「无月经」行。 |

### 3.4 精灵类：现实 28 日列段 + 除月经 7 日外全盘 ×2 → **49** 日一环

**适用范围**：**全部精灵类标签共用同一套日历**（高等精灵、半精灵、暗精灵等 **无支系分轨**；若半精灵需弱化，再以 `TALENT` 或系数表减成，**边界仍按 §3.4**）。

| 项目 | 定案 |
|------|------|
| **规则（一句话）** | 先列出 **现实女性一环 28 日** 各阶段日数（须符合 §3.0 顺序）；**月经段固定 7 日且不加倍**；**月经以外的每一段**日数 **全部 ×2**。**精灵总长 = 7 + (28−7)×2 = 7+42 = 49** 游戏日，**`cycle_length` = 49**（模 49）。 |
| **现实一环（示意拆段）** | 与 §3.0 顺接的 **示例**（各段之和须为 **28**，且 **月经=7**）：**月经 7** → **经后卵泡 8** → **危险／易孕 6** → **黄体 7**（7+8+6+7=28）。此为叙事与妊娠脚本的 **边界占位**；若策划改某一非月经段的基底日数，**须同步调整其余非月经段** 使总和仍为 **21**（28−7）。 |
| **精灵一环（同例 ×2）** | **月经 7**（不变）→ 经后卵泡 **16** → 危险 **12** → 黄体 **14**；**7+16+12+14=49**。排卵／易孕判定按 **49 日模** 重划 `[start,end)`。 |
| **说明** | **不以 §3.1 的 15 日短环**为精灵基底。此前文档中 **46／47** 等拆法作废，以本条 **49** 为定案总长。 |

### 3.5 类人族（与人类共用 §3.1）

下列种族**不单独造日历**，实现时归入 **`MENSTRUAL_RULE_HUMAN_15`**（名称示意）即可：

- **矮人族、半身人**（与半身人同类的「小人族」分支若存在，策划可并入本行同一规则）  
- **半吸血鬼**（纯血吸血鬼无周期，见 §1 第 3 条）  
- **半兽人** 等以 **人类混血／人类型月经解剖** 为默认叙事的种族  
- 其他策划指定为「**参照人类即可**」的类人标签

**例外**：带 **精灵／半精灵／暗精灵** 等血统者 **优先 §3.4**（精灵加倍轨），**不与 §3.1 类人混用**。

### 3.6 种族规则矩阵（摘要）

| 种族/标签（示例） | 周期模板 | 备注 |
|-------------------|-----------|------|
| **人类**（魔界） | §3.1，15 日；症状可减半 | 环状排期：经 3、易孕 4 等。 |
| **类人族**（矮人、半身人、半吸血鬼、人类侧半兽人等） | **同 §3.1** | 与人类同一 `cycle_length`。 |
| **魔族、天使、天人** | **§3.2**，30 日 | **彼此共用**；女神不在此列。 |
| **龙人、翼人、动物类亚人**（兽耳等） | **§3.3** | 年 **2** 次月经各 **7** 日；**30** 日子环（普通 **20** / 发情 **10**）；发情易湿易孕，普通日同房不怀孕。 |
| **精灵类**（高等、半精灵、暗精灵等 **同一规则**） | **§3.4** | 现实 **28** 日列段（**月经 7**）；**除月经外各段 ×2** → **49** 日一环（见 §3.4）。 |
| **仙人、女神、纯吸血鬼、妖狐** | **无月经** | 与 §1 第 3 条一致；自控／神力、血统或妖力设定；不推进月经计数。**兽耳族**中仅当种族判定为 **妖狐** 时走本行；猫耳、犬耳等走 **§3.3**。 |

**种族判定落点（实现时查阅）**：优先复用现有 **`TALENT`／角色作成 `CSTR`／种族 BIT**；若无单一字段，再用 **`CFLAG` 位图「规则覆盖 ID」**（避免在文档里写死未存在的 ID）。

## 4. 与现有/未来系统的接口（仅清单）

- **妊娠**：经期是否禁止部分 COM、是否暂停排卵逻辑、魔界人类「减半」是否影响受孕率 —— **需单独 CONFIG 或子规则**。  
- **泌乳**：与 **药催／魔催** 线并存时，建议 **发情期** 微调喷乳欲望、**生理期** 仅轻量影响体力与痛阈，避免与 `MARK:母乳体质` 强耦合。  
- **调教 SOURCE**：阶段 → `TIMES` 痛苦/耻辱/淫欲 等 **分渠道** 修正；**封顶**写在统一 `#FUNCTION`。  
- **排泄羞耻**：腹痛与膀胱不适 **叙事分流**（已实现膀胱链的可接「经前腹胀」占位）。  
- **榨乳口便器线**：贞操带／尿道塞在特定阶段是否 **CONFIG 禁止** —— 远期再接。

## 7. 与当前代码占位的关系

- 现有 **`RANCHORAL_MENSTRUAL_DAILY_TICK`** 仅为 **模 28 均匀日进**，**未**体现种族；日后应改为 **`TRYCALL` 统一入口** 内按种族 **`cycle_length`（15 / 30 / 49 等）**、**§3.3（年度双月经 + 模 30 子环）**、或 **§3.4 精灵（模 49）** 分别推进。  
- **`CFLAG:939`** 在「无周期」种族应 **不递增** 或恒置某一哨兵值，避免成就/统计误读；实现时或改名为「周期内日索引」以免与旧档语义混淆。
